# vfox-docker-health

[![CI](https://github.com/jbox-web/vfox-docker-health/workflows/CI/badge.svg)](https://github.com/jbox-web/vfox-docker-health/actions)

A plugin for the [mise version manager](https://github.com/jdx/mise) to install [docker-health](https://github.com/jbox-web/docker-health)

## Install

```
mise plugin install docker-health https://github.com/jbox-web/vfox-docker-health.git
```
